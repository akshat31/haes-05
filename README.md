HAES Institute — React (Vite) site

This repository uses a Vite + React static site.

Run locally:

1. npm install
2. npm run dev

Build and preview:

1. npm run build
2. npm run preview

Deploy to GitHub Pages:

1. Set "homepage" in package.json to https://<username>.github.io/<repo>
2. npm run build
3. npm run deploy
